# TEA -- Trastorno de espectro autista

## PARA LEVANTAR DOCKER
	
	docker start tea-db

## PARA SABER SI SE ESTA ANDANDO
	
	docker ps

## PARA VER LA IP A LA CUAL CONECTAR
	
	docker inspect tea-db

## PARA DETENER EL CONTENEDOR
	
	docker stop tea-db

## IMPORTANTE
	
	No hacer "docker run --"
	No hacer "docker rm tea-db"
